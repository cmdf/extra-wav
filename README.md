# owav

Play PCM WAV audio in Windows Console.


## usage

```batch
> owav [-i|--input <input wav file>]

:: [] -> optional argument
:: <> -> argument value
```

```batch
:: play council.wav file
> owav --input council.wav

:: play council.wav, using stdin
> cat council.wav | owav
```
